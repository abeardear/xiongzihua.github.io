Hexo搭建github博客

发表于 2019-11-13 分类于 hexo 阅读次数：

基于hexo搭建github博客网站，并托管保存用于生成网页的hexo配置和源代码

准备知识

  • git
  • hexo

hexo是一个用于快速搭建博客的框架，是npm(Node Package Manager)的包，而npm与nodejs相辅相成，具体可以参见链接

对于hexo，我们只求知道基本使用和配置即可

初始化博客仓库

完成后初始化后会在blog文件夹产生hexo相关内容，这些内容是用于生成静态网页的源码，我们使用时只需要做相关配置即可

文件/文件夹 说明
_config.yml 配置文件
public 生成的静态文件，这个目录最终会发布到服务器
scaffolds 一些通用的markdown模板
source 编写的markdown文件，_drafts草稿文件，_posts发布的文章
themes 博客的模板

github托管

1.首先，我们需要在github创建 username.github.io 的仓库，这个仓库是个空项目
2.hexo发布的静态网页默认在master分支
3.可以将生成静态网页的hexo配置代码，托管在 username.github.io 仓库的另一个分支，例如source分支，这样就可以保证不丢失辛苦的配置了

hexo配置简介

hexo的_config.yml

  • title: 博客标题
  • author: 作者名称
  • deploy: 配置部署信息

更多信息可以自由探索

hexo命令简介

hexo g 生成静态页面
hexo s 在本地开启服务
hexo clean 清空缓存
hexo d 远程部署
